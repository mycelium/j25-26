# Простой JSON парсер на Java

Библиотека для парсинга JSON строк в объекты Java и обратной сериализации. 

## Сборка и запуск

1. Откройте терминал и перейдите в папку с проектом:

    MacOS/Linux:
    ```
    cd tasks/term-2/1
    ```
2.  Компиляция исходных файлов

    MacOS/Linux:
    ```
    javac jsonlib\*.java Main.java
    ```
3. Запуск программы:
    ```
    java Main
    ```
