# Лабораторная работа №6 Java: K-Nearest Neighbors (KNN) Classifier
# Задание: Реализовать приложение классификации точек методом ближайших соседей (KNN)

## Алгоритм K-NN

1. **Вычисление расстояний**: Для заданной тестовой точки вычисляются Евклидовы расстояния до всех точек обучающей выборки
   ```
   distance = √[(x₁-x₂)² + (y₁-y₂)²]
   ```

2. Выбор K ближайших соседей: Из всех точек выбираются K ближайших (в данной реализации K=3)

3. Категория определяется по большинству голосов среди K ближайших соседей

4. Возвращается наиболее часто встречающаяся категория


После выполнения программа создаёт 5 PNG-файлов (`graph_1.png` - `graph_5.png`), каждый из которых показывает:
- Все обучающие точки с цветовой кодировкой по категориям
- Тестовую точку с чёрной обводкой и цветом предсказанной категории
- Координатную сетку
- Легенду с обозначениями категорий

Используется 3 категории точек:
- **Категория A** (фиолеторая): точки в нижней левой области (координаты ~1-3)
- **Категория B** (розовая): точки в верхней правой области (координаты ~7-9)
- **Категория C** (голубая): точки в центральной области (координаты ~3-7)
