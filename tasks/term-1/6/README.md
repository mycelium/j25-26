# Лабораторная работа: KNN Классификатор

## Описание
Данный проект реализует классификатор K-ближайших соседей (KNN) на Java без использования сторонних библиотек машинного обучения.
Приложение генерирует синтетический набор данных из 2D точек, принадлежащих 3 разным классам, обучает классификатор и классифицирует новую точку.
Результат визуализируется и сохраняется в файл `knn_result.png`.

## Требования
- Java 23 (или совместимая версия JDK)
- Gradle

## Запуск

Выполните следующую команду в терминале:

```bash
./gradlew run
```

Эта команда выполнит следующие действия:
1. Скомпилирует проект.
2. Запустит приложение.
3. Выведет результат классификации в консоль.
4. Сгенерирует изображение `knn_result.png` в папке `app`.

## Структура проекта
- `App.java`: Точка входа. Генерирует данные и запускает демонстрацию.
- `KNNClassifier.java`: Реализация алгоритма KNN (Евклидово расстояние, голосование).
- `Plotter.java`: Утилиты для визуализации данных и результата классификации.
- `Point.java`: Класс данных, представляющий 2D точку с меткой класса.
