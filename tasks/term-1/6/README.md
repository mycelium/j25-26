# Описание
Приложение реализует классификацию двумерных точек методом K ближайших соседей (KNN) без использования специализированных библиотек.

На вход подаётся точка без класса.  
На основе расстояния до ближайших обучающих точек определяется её класс.

Также строится изображение с визуализацией точек разных классов и новой классифицируемой точки.

# Алгоритм

Используется алгоритм KNN (K-Nearest Neighbors).

Основные шаги:

1. Для новой точки вычисляется расстояние до всех точек.
2. Точки сортируются по расстоянию.
3. Выбираются K ближайших соседей.
4. Класс определяется по большинству голосов среди соседей.

Расстояние считается по евклидовой метрике:

distance = sqrt((x1 - x2)^2 + (y1 - y2)^2)

# Результат работы

Программа:

1. выводит предсказанный класс новой точки
2. сохраняет изображение с точками

Пример вывода:

```
Scenario 0
New point: (3.2, 8.1)
Predicted class: 3
```

Создаётся файл:

```
knn_plot_0.png
```

На изображении:
- точки разных классов отображаются разными цветами
- новая точка отображается крупнее
- подписан предсказанный класс
