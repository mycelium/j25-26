# Лабораторная работа: Классификатор KNN

Этот проект реализует простой классификатор K-ближайших соседей (KNN) на Java для классификации двумерных точек по трем классам (A, B, C). Также генерируется визуализация данных и результата классификации.

## Требования
- Java 21
- Gradle 8.x (обертка включена в проект)

## Как запустить
Проект использует Gradle для сборки и запуска.

Для запуска приложения выполните:
```bash
./gradlew run
```

Это выполнит следующие действия:
1. Сгенерирует синтетические обучающие данные.
2. Классифицирует тестовую точку (5.0, 5.0).
3. Выведет результат классификации в консоль.
4. Сохранит график визуализации в файл `knn_result.png`.

## Структура проекта
Исходный код находится в пакете `org.example` по пути `app/src/main/java/org/example/`:
- `App.java`: Точка входа и демонстрационная логика.
- `KNNClassifier.java`: Реализация алгоритма KNN.
- `Point.java`: Структура данных для двумерных точек с метками.
- `Plotter.java`: Утилита для генерации PNG-графика.

## Запуск тестов

Для запуска всех тестов выполните:
```bash
./gradlew test
```

Для просмотра подробного вывода с именами тестов:
```bash
./gradlew clean test
```

Тесты находятся в `app/src/test/java/org/example/KNNClassifierTest.java` и включают:
- Базовую классификацию (k=1, голосование большинством)
- Вычисление расстояний (отрицательные, дробные, большие координаты)
- Различные значения k
- Граничные случаи (одна точка, точки в одном месте)
- Обработку ошибок (null, пустые данные, недопустимые k)
- Тесты производительности (до 10000 точек)

## Примечания
- Внешние зависимости не используются; проект опирается на стандартную библиотеку Java.
- Проект использует стандартную структуру директорий Gradle.
