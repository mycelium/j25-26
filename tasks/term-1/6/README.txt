В коде представлена реализация алгоритма k-nearest neighbors для двумерных точек без сторонних библиотек:
каждая точка описывается классом `Point` с полями `x`, `y` и номером класса `clazz`,
классификатор `KnnClassifier` ищет k ближайших соседей и выбирает наиболее часто встречающийся класс,
а в `Main` задаётся обучающая выборка минимум из трёх классов (0, 1, 2), создаётся новая точка,
для неё считается класс и результат печатается в консоль.

Для построения графика используется класс `SimplePlotter`, который рисует все точки разных классов разными цветами
и сохраняет картинку в файл `knn_plot.png` в папке `app`.

Результат:
На графике видно три кластера точек разных классов: красный, зелёный и синий.
Новая точка сначала не имеет класса, её координаты попадают примерно между кластерами,
и алгоритм KNN присваивает ей тот класс, чьи точки расположены к ней ближе всего;
именно эта уже «раскрашенная» новая точка добавляется к общему набору и тоже отображается на изображении.

Для запуска нужны Java 17+ и Gradle, запускается командой `./gradlew run`,
после запуска в консоли появятся координаты новой точки и её предсказанный класс,
а в папке будет создан файл `knn_plot.png`.
