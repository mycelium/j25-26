Элементы множества  хранятся не в памяти в явном виде, а в  функциях-предикатах. 
Множество - это функция, которая для  отвечает true, 
если целое число принадлежит множеству и false - если нет.
Пример:
val set = singletonSet(1)  // создает функцию (x: Int) => x == 1
contains(set, 1)           // просто вычисляет set(1) и возвращает результат - true
contains(set, 2)           // вычисляет set(2) и возвращает результат false
