В данной лабораторной работе тип Set, представляющий множество, является не структурой данных, а условным предикатом, который принимает на вход целое число и возвращает, True, если это число принадлежит множеству, заданному этим предикатом, и False, если не принадлежит. 

Элементы множества хранятся не в памяти, а в коде условного предиката. Для любого числа можно вызвать метод contains, который определит, принадлежит число множеству или нет. Также подобная реализация множества позволяет создавать бесконечные структуры.
