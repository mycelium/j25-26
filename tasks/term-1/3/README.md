Элементы множеств ане хранятся явно в памяти. Множество - это функция `(Int => Boolean)`, а не структура данных. Например, множество из числа 1 - это просто функция, которая спрашивает: "Это число равно 1?" Если да - значит число в множестве, если нет - не в множестве. Все операции создают новые функции-правила, но сами числа ни в каких списках или массивах не сохраняются.
