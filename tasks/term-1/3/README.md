В данной реализации множество представлено как функция типа Set = Int => Boolean. Такая функция для каждого целого числа указывает, содержится ли этот элемент в множестве (возвращая true или false).

Множество задается через характеристическую функцию, которая для любого входного значения проверяет его принадлежность множеству. По сути, множество — это условие, которому должны удовлетворять элементы.

Таким образом, элементы хранятся неявно в виде логического выражения (функции), которое определяет множество. Для операций объединения, пересечения, разности и фильтрации создаются новые функции — композиции исходных характеристических функций.
