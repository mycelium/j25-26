Элементы не хранятся явно — они существуют в виде логики функции.
Множество {1, 2, 3} — это функция, возвращающая true для 1, 2, 3 и false для остальных.
Так можно представить даже бесконечные множества: (x: Int) => x % 2 == 0 описывает все четные числа.

## Запуск тестов

Для запуска всех тестов:
```bash
.\gradlew.bat test
```

Для просмотра подробного вывода с именами тестов:
```bash
.\gradlew.bat test --rerun-tasks
```

Примечание: Первый запуск загружает зависимости Scala и может занять несколько минут.
