
# Ответ на вопрос: где хранятся элементы множества?

В данной реализации множества **не хранят элементы явно** в виде структур данных (массивов, списков и т.д.). Вместо этого множество представлено как **функция-предикат** типа `Int => Boolean`.

- **Множество** — это функция, которая принимает целое число и возвращает `true`, если элемент принадлежит множеству, и `false` в противном случае
- **Пример**: множество {1, 2} представляется как:
  ```scala
  val s = (x: Int) => x == 1 || x == 2
  ```
