FunSets — функциональные множества целых чисел в Scala.

Каждое множество представлено функцией типа Int => Boolean.
Эта функция проверяет, принадлежит ли элемент множеству.

Основные операции:
- singletonSet(x) — множество с одним элементом x
- union(s, t) — объединение множеств
- intersect(s, t) — пересечение множеств
- diff(s, t) — разность множеств
- filter(s, p) — фильтрация элементов по условию p
- forall(s, p) — проверяет, что все элементы множества удовлетворяют условию p
- exists(s, p) — проверяет, что хотя бы один элемент множества удовлетворяет условию p
- map(s, f) — применение функции f ко всем элементам множества

Примеры использования:
- {1,2,3} после filter x % 2 == 1 → {1,3}
- {1,2,3} после map x => x*x → {1,4,9}
