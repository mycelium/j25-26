## 3. Scala: Functional Set
### Хранение элементов множества
В задании множество представлено как функция типа `Int => Boolean`, которая 
принимает целое число (элемент) и возвращает `true`, если элемент  принадлежит множеству, и 
возвращает `false` в противном случае. То есть множество представлено не 
как коллекция элементов, а как правило, по которому определяется принадлежность 
элемента к множеству. Получается, что сами элементы нигде явно не хранятся.
