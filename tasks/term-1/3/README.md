# HSAI 25-26 Java course - 1
## Лаб3. Scala: Functional Set

Напрямую мы нигде не храним список элементов. За определение "принадлежности" числа множеству отвечает характеристическая функция.
    
Стоит отметить, что данный подход эквивалентен следующему способу создания множеств в математике: $\{ x \in \mathbb{R}, x\hspace{-1mm}\mod 2\equiv 0 \}$ - множество рациональных четных чисел.

Такой подход позволяет не только хранить множества любого размера (в том числе и теоритически бесконечные множества), но и делать это максимально эффективно: достаточно задать характеристическую функцию.
