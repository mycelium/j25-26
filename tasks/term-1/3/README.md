Сами числа нигде не хранятся, мы сохраняем предикаты, которые мы проверяем для конкретного числа, вызывая метод contains.

Это позволяет, в том числе, реализовывать бесконечные структуры, например, всех четных чисел: (x: Int) => x%2 == 0.
Конечно, все четные числа тут не хранятся, но если мы вызовем contains(set, 2) $\rightarrow$ 2%2 == 0 $\rightarrow$ true, то есть элемент содержится в множестве.
А при contains(set, 3) $\rightarrow$ 3%2 == 0 $\rightarrow$ false, то есть такого элемента нет.
