## Где хранятся элементы множества?

Элементы множества не хранятся в явном виде в памяти, в данной реализации множество - это функция, которая для любого целого числа отвечает true, если число принадлежит множеству и false - если нет.

Элементы хранятся в логике функций-предикатов, а не в явных структурах данных.

### Пример:
```scala
val set = singletonSet(1)  // создает функцию (x: Int) => x == 1
//функция знает про элемент 1, но не хранит его в структуре данных
contains(set, 1)           // просто вычисляет set(1) и возвращает результат - true
contains(set, 2)           // вычисляет set(2) и возвращает результат false
