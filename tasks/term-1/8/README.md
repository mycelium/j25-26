# MNIST CNN Classification

Лабораторная работа по классификации рукописных цифр MNIST с использованием сверточной нейронной сети (CNN) на базе DeepLearning4j.

## Запуск

### Требования
- Java 23+
- Gradle 8.14.3+

### Сборка и запуск
```bash
# Сборка проекта
./gradlew build

# Запуск приложения
./gradlew run
```

Приложение автоматически:
1. Загрузит датасет MNIST
2. Построит сверточную нейронную сеть
3. Обучит модель на тренировочных данных
4. Оценит точность на тестовых данных

## Архитектура сети
- 2 сверточных слоя с ReLU активацией
- 2 слоя субдискретизации (MaxPooling)
- Глобальный пулинг слой
- Полносвязный слой
- Выходной слой с Softmax

## Технологии
- DeepLearning4j - фреймворк для глубокого обучения
- ND4J - библиотека для работы с многомерными массивами
- Java 23
