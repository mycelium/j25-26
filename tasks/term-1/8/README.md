# Классификация изображений с использованием DeepLearning4j

Проект реализует сверточную нейронную сеть (CNN) для распознавания рукописных цифр на основе датасета MNIST.

Для корректной работы предсказания на пользовательских изображениях необходимо:

1. Подготовить изображение с цифрой (28×28 пикселей, черно-белое)
2. Указать полный путь к файлу в коде:
```java
String testImagePath = "C:/полный/путь/к/изображению/цифра.png";
```
По умолчанию программа ищет файл "3.png" в рабочей директории. Для использования собственных изображений необходимо изменить путь в коде.


## Формат входных изображений
- Размер: 28×28 пикселей
- Цвет: оттенки серого
- Фон: темный, цифра светлая
- Формат файла: PNG
