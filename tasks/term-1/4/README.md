Программа автоматически определяет размер файла. Для файлов размером до MAX_FILE_SIZE_FOR_FULL_LOAD она загружает весь файл в память. Для файлов большего размера (> MAX_FILE_SIZE_FOR_FULL_LOAD) она использует потоковую обработку (построчно). При недостатке памяти она автоматически переключается на потоковую обработку.

Укажите путь к файлу в коде:
```java
Path filePath = Path.of("путь_к_вашему_файлу.txt");
```
