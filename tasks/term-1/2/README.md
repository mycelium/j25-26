# Лабораторная работа 2: Рекурсивные функции

## Описание

Лабораторная работа по рекурсивным функциям на Scala.

## Упражнения

1. **Pascal's Triangle** - Вычисление элементов треугольника Паскаля
2. **Parentheses Balancing** - Проверка сбалансированности скобок
3. **Count Change** - Подсчет способов разменять сумму монетами
4. **N-Queens** - Задача о N ферзях

## Требования

- Scala 2.13+

## Запуск

Для компиляции и запуска:
```bash
.\gradlew.bat run
```

## Запуск тестов

Для запуска всех тестов:
```bash
.\gradlew.bat test
```

Для просмотра подробного вывода с именами тестов:
```bash
.\gradlew.bat test --rerun-tasks
```

Примечание: Первый запуск загружает зависимости Scala и может занять несколько минут.

Тесты находятся в `src/test/scala/recfun/MainTest.scala` и включают:
- Pascal's Triangle (9 тестов) - базовые случаи, края, середина
- Parentheses Balancing (15 тестов) - сбалансированные и несбалансированные
- Count Change (11 тестов) - различные суммы и наборы монет
- N-Queens (9 тестов) - размеры от 0 до 8
