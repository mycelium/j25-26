Оптимизация времени выполнения перемножения матриц заключается в транспонировании второй матрицы. 
При перемножении больших матриц происходит частое обращение к данным двумерного массива, обход строки матрицы происходит быстро, но вот обход столбца занимает больше времени за счет хранения данных в памяти.
Если вторую матрицу транспонировать, то нужно будет бегать не по столбцам а по строкам, что дает значительные изменения в выполении программы. 

Перемножения двух матриц 1000х1000 дают следующие результаты:  
```aiignore
Время выполнения: 4210 мс
Время выполнения с оптимизацией: 1147 мс
```
