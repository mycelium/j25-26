# Параллельное умножение матриц (MatrixMultPar)

Проект содержит два метода умножения матриц:

1. **`multiply()`** - однопоточная реализация из первой лабораторной (оптимизированная версия)
2. **`multiplyParallel()`** - многопоточная реализация 

Подбор оптимального количества потоков (число ядер -4, число логических процессоров - 8)
Тестирование проводилось на матрицах размером **1000x1000** с усреднением по 10 запускам.

Минимальное время достигнуто при 7 потоках (84 мс) - это оптимальное значение для данной платформы.
Параллельная реализация с одним потоком (464 мс) работает медленнее однопоточной (450 мс) из-за создания потоков и синхронизации.
