В данной работе реализованы две версии умножения квадратных матриц: однопоточная (из работы 1) и многопоточная. 
Многопоточная версия построена на основе пула потоков ExecutorService, где строки результирующей матрицы равномерно распределяются между потоками.
Тестирования, выполнявшиеся для матриц размеров 100×100, 500×500 и 1000×1000,показывают, что многопоточная реализация даёт сильный выигрыш по времени по сравнению с однопоточной:
для маленьких матриц ускорение не сильно заметно в виду накладных расходов, но уже для матриц 500×500 и 1000×1000 результы ощутимы.
При использовании 4 потоков многопоточная реализация вычисляет в 3–4 раза быстрее, чем однопоточная. Подбор оптимального количества потоков в работе осуществляется путем тестирования,
я остановилась на реализации с 1, 2 и 4 потоками.
