# 7. Java: Parallel matrix multiplication

Лабораторная работа была выполнена на устройстве с процессором 13th Gen Intel(R) Core(TM) i5-13500H с 16 логическими ядрами.

Оптимальное количество потоков для умножения матриц большой размерности было определено в результате тестирования производительности при перемножении матриц различного размера с использованием различного количества потоков: 4, 8, 12, 16. 

Результат тестирования перемножения матриц размером 500x500:
| Количество потоков | Время выполнения |
|--------------------|------------------|
| 4   		     | 11 мс            |
| 8 		     | 8 мс             |
| 12 		     | 8 мс             |
| 16		     | 8 мс             |

Результат тестирования перемножения матриц размером 1000x1000:
| Количество потоков | Время выполнения |
|--------------------|------------------|
| 4   		     | 72 мс            |
| 8 		     | 58 мс            |
| 12 		     | 56 мс            |
| 16		     | 41 мс            |

Результат тестирования перемножения матриц размером 1500x1500:
| Количество потоков | Время выполнения |
|--------------------|------------------|
| 4   		     | 331 мс           |
| 8 		     | 232 мс           |
| 12 		     | 236 мс           |
| 16		     | 210 мс           |

Результат тестирования перемножения матриц размером 2000x2000:
| Количество потоков | Время выполнения |
|--------------------|------------------|
| 4   		     | 1170 мс          |
| 8 		     | 797 мс           |
| 12 		     | 666 мс           |
| 16		     | 559 мс           |

Тесты показали, что чем больше потоков используется, тем лучше производительность. Таким образом оптимальное количество потоков для работы с матрицами большой размерности - 16, что соответствует количеству логических ядер процессора.
