#Параллельное умножение матриц.
Ноутбук Samsung Galaxy Book2 содержит процессор Intel Core i7 12700.
Этот процессор имеет 20 потоков (https://www.citilink.ru/product/processor-intel-core-i7-12700-lga-1700-oem-cm8071504555019-srl4q-1795642/)
значит, более 20 потоков не смогут работать параллельно на этом ноутбуке и будут терять время из-за передачи потока компьютера между потоками программы.
Программа распараллелена таким образом, что каждая трока матрицы вычисляется отдельным потоком, значит создавать потоков больше, чем строк в матрице, тоже не имеет смысла.
=> threadsnum = Math.min(A.length(), 20)
